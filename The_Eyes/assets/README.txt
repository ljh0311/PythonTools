# The Eyes - Assets Directory

This directory contains assets used by the application.

## Contents

- `icon.ico`: Application icon file

## Notes

The icon file should be a valid .ico file. If you want to use a custom icon:

1. Create a 32x32 or 64x64 pixel image
2. Convert it to .ico format
3. Save it as `icon.ico` in this directory

You can use online conversion tools or software like GIMP to create .ico files.

## Default Icon

If the icon.ico file is not present, the application will use the default tkinter icon. The application will still work correctly without custom icons.
