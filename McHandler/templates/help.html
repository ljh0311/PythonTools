Help & Documentation

About CDID Car Tuning Assistant

The CDID Car Tuning Assistant is an AI-powered tool that helps you tune cars in CDID (a Roblox car tuning experience). It uses Ollama AI to provide intelligent tuning suggestions and problem diagnosis.

Features:
  • Tuning Suggestions: Get AI-powered recommendations for engine and suspension tuning based on your goals
  • Problem Diagnosis: Describe issues with your car's performance and get specific solutions
  • Ollama Integration: Uses local AI models for privacy and offline capability
Tuning Suggestions Mode

Use this mode when you want to improve your car's performance or achieve specific goals.

How to use:
  1. Select "Tuning Suggestions" mode
  2. Describe your car and its current characteristics
  3. Specify your tuning goals (e.g., better handling, more speed, drift setup)
  4. Select focus areas (Engine, Suspension, or both)
  5. Click "Get Tuning Suggestions"
Example:
Car Description: "I have a high-powered sports car that accelerates quickly but struggles in corners"
Tuning Goals: "I want better cornering stability and more predictable handling"
Problem Diagnosis Mode

Use this mode when your car is experiencing specific problems or issues.

How to use:
  1. Select "Problem Diagnosis" mode
  2. Describe the problem you're experiencing
  3. (Optional) Include your current tuning settings
  4. Click "Diagnose Problem"
Example:
Problem Description: "My car oversteers too much in corners and feels unstable at high speeds"
Current Settings: "Suspension stiffness: 0.9, Engine power: 600hp"
Ollama Setup

This application requires Ollama to be installed and running on your computer.

Installation:
  1. Download and install Ollama from https://ollama.ai
  2. Start Ollama service: ollama serve
  3. Download a model: ollama pull llama3.2
Recommended Models:
  • llama3.2 - Good balance of speed and quality (recommended)
  • llama3.2:3b - Faster, smaller model
  • mistral - Good for reasoning tasks
The application will check Ollama connection status automatically. Make sure Ollama is running before using AI features.
Tips
  • Be specific in your descriptions for better results
  • Include current settings when diagnosing problems
  • Test changes incrementally in CDID
  • Different models may provide different perspectives
Common Tuning Parameters
Engine:
  • Power/Torque
  • Gear ratios
  • RPM limits
Suspension:
  • Stiffness
  • Damping
  • Ride height
